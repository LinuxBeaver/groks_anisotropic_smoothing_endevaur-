INSTALL
Place the ggr files in your gradient folder which in Windows XP is usually here
C:\Documents and Settings\USERNAME\.gimp-2.6\gradients
Where USERNAME is your user name folder

OR you can place them directly into the Windows XP Gimp program directory which is typicly here..
C:\Program Files\Gimp-2.6\share\gimp\2.0\gradients  folder
